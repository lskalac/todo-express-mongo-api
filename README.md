# todo-express-mongo-api

This is the Todo Rest Api written in the following technologies:
* NodeJS (TS)
* Express
* MongoDB

Steps to start the app:
* npm install
* create .env file (find below the varibales)
* npm start

.env : <br>
NODE_ENV=development
HOST=localhost <br>
PORT=7000 <br>
DB_NAME=Todo <br>
DB_USER= <br>
DB_PASSWORD= <br>
